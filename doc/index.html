Introduction项目介绍

What is Premium Address? 什么是 Premium Address？

Premium Address is a decentralized mining protocol that turns computational work into digital collectibles. Miners run software to search for blockchain addresses containing long, unbroken runs of identical digits — for example, an address ending in ...9999999999. Premium Address 是一个去中心化挖矿协议，将算力转化为数字收藏品。矿工运行软件，搜索包含长串相同数字的区块链地址——例如一个以 ...9999999999 结尾的地址。

Each valid discovery triggers three things simultaneously: a unique NFT is minted with fully on-chain artwork, ADD tokens are rewarded to the miner, and a real smart contract wallet is deployed at that vanity address. The longer the streak, the rarer the find. 每一次有效发现会同时触发三件事：铸造一个拥有完全链上艺术品的独特 NFT、向矿工发放 ADD 代币奖励、并在该靓号地址上部署一个真正的智能合约钱包。连续位数越长，发现越稀有。

Gameplay玩法

How Mining Works 挖矿流程

1
Search搜索
Run the miner software to search for blockchain addresses with long runs of identical digits. The miner tests billions of combinations until a qualifying pattern is found.运行矿工软件，搜索包含长串相同数字的区块链地址。矿工会测试数十亿种组合，直到找到符合条件的模式。
2
Submit提交
When a valid pattern is found (minimum 9 consecutive identical digits), submit the discovery on-chain. Batch submissions are supported — multiple finds in a single transaction.当发现有效模式（至少 9 位连续相同数字）时，将发现提交到链上。支持批量提交——单笔交易可包含多个发现。
3
Mint & Earn铸造与获利
The contract verifies the pattern, deploys a smart wallet at the vanity address, registers the NFT, and rewards you with ADD tokens — all in one atomic transaction.合约验证模式，在靓号地址上部署智能钱包，注册 NFT，并奖励 ADD 代币——所有操作在一笔原子交易中完成。
4
Claim Ownership认领所有权
Optionally claim ownership of the address NFT by paying a small BNB fee (starting at 0.001 BNB). This grants you full control over the smart wallet and the NFT.可选择支付少量 BNB 费用（起价 0.001 BNB）认领地址 NFT 的所有权。这将赋予你对智能钱包和 NFT 的完全控制权。

⚠ Longer streaks are exponentially harder to discover. Finding a 10-digit streak is roughly 16× harder than a 9-digit streak. A 14-digit streak? Over 65,000× harder. This makes higher-tier NFTs genuinely scarce. ⚠ 更长的连续位数呈指数级难度增长。发现 10 位连续比 9 位难约 16 倍。14 位连续？超过 65,000 倍。这使得高品阶 NFT 真正稀缺。
